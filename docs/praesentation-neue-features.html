Was ist MCP?

Model Context Protocol — ein offener Standard von Anthropic.

KI-Modelle können von sich aus nicht auf externe Systeme zugreifen. MCP definiert ein Protokoll, über das sie Tools aufrufen und Aktionen ausführen.
  • MCP-Server stellt Tools bereit
  • MCP-Client (Claude, Cursor, ...) entdeckt sie automatisch
  • Nutzer tippt natürliche Sprache → KI ruft passende Tools auf

MCP-Autorisierung

OAuth 2.1 via Supabase — kein manuelles Token-Kopieren.

  1. MCP-Server-URL in Cursor eintragen
  2. Cursor erkennt den OAuth-Server automatisch
  3. Browser öffnet sich → Supabase-Login
  4. Nutzer bestätigt Freigabe auf /connect
  5. Cursor erhält Token → alle Tools sofort bereit

Funktioniert mit Cursor, Claude und jedem MCP-fähigen Client.
